提示：本地SSH客户端与本地SFTP客户端命令参数变量意义如下
  • {login_host}  SSH 连接 IP
  • {login_user}  SSH 连接用户
  • {login_passwd}  SSH 连接密码
  • {port}  SSH 连接端口
  • {username}  后端真实主机用户名（用于设置客户端窗口标题显示）
  • {host}  后端真实主机 IP（用于设置客户端窗口标题显示）
  • {hostname}  后端真实主机名（用于设置客户端窗口标题显示）
